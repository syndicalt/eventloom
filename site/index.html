Event-sourced runtime for AI agents.

Eventloom turns agent execution into an append-only event log with validated actor intentions, deterministic replay, tamper-evident integrity, MCP tooling, and optional Pathlight and HALO export.

Get started npm package

Event log

Tamper-evident agent history

Every accepted event is sealed into a JSONL hash chain. You can append external human approvals, replay old runs, and verify that runtime history still matches the recorded contents.

Runtime contract

Actors emit intentions, not mutations

Actors receive mailbox items and submit structured intentions. The orchestrator validates them before anything reaches the log, keeping workflow state explainable and auditable.

Replay

Deterministic projections

Task, research, effect, and actor mailbox state rebuild from the log. A replay gives you event counts, integrity status, projections, and a deterministic projection hash for comparison.

How it runs

A small runtime loop with inspectable state.

Eventloom is intentionally boring infrastructure: local files, typed events, deterministic reducers, and a CLI you can run before adding databases, queues, or dashboards.

Install

Start from npm.

CLI

npm install @eventloom/runtime
npx eventloom run software-work /tmp/eventloom.jsonl
npx eventloom replay /tmp/eventloom.jsonl

TypeScript

import { createRuntime } from "@eventloom/runtime";

const runtime = createRuntime("/tmp/eventloom.jsonl");
await runtime.runBuiltIn("research-pipeline");

const replay = await runtime.replay();
console.log(replay.integrity.ok);

HALO

npx eventloom export halo .eventloom/agent-work.jsonl \
  --out eventloom-halo-traces.jsonl \
  --project-id eventloom

MCP

npx @eventloom/mcp --root .

Agent integration

Teach coding agents to leave a replayable trail.

Eventloom now includes a repo-local Codex skill for recording agent goals, task lifecycle events, handoffs, verification, and replay summaries into a local JSONL log. The MCP package exposes the same local log operations to editors and agent clients over stdio, including rebuilt actor mailboxes through eventloom_mailbox and HALO JSONL export through eventloom_export_halo.

Optional bridge

Export runs to Pathlight and HALO.

Eventloom can translate actor turns and related runtime events into Pathlight traces, spans, and span events for visual inspection. It can also project local Eventloom logs into HALO-compatible OpenTelemetry JSONL for agent failure-mode analysis.
